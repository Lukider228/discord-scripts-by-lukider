# 🌟 Discord Scripts by Lukider

Набор премиальных юзерскриптов для Discord, созданных с чистотой, удобством и стилем. Просто установи — и вот уже твой дискорд-опыт улучшен!

---

## 📦 Что здесь?

### 🗑️ **Discord Mass Deleter** — Массовое удаление сообщений (v2.1)

Когда нужно быстро очистить свои сообщения на сервере, в ЛС или канале — вот твой инструмент.

**Основа:** Переработка популярного скрипта [victornpb/deleteDiscordMessages](https://github.com/victornpb/deleteDiscordMessages) с премиальным редизайном, оптимизацией и полным русским интерфейсом.

**Что умеет:**
- 🎯 Удалять только **твои сообщения** или сообщения конкретного автора
- 📅 Фильтровать по датам (удалить сообщения только за определённый период)
- 🔗 Фильтровать по содержимому (текст, ссылки, файлы)
- 👀 **Холостой запуск** — сначала посмотри, что удалится, без риска
- 🛡️ Умная обработка архивных веток (автоматически пропускает заблокированные каналы)
- ⚡ Супер-оптимизирована — не будет тормозить твой браузер

**Особенности в v2.1:**
- 🌹 **Glassmorphism интерфейс** с розовым и фиолетовым градиентом
- 📍 Боковая панель шириной 500px, закрепленная на экране
- 🇷🇺 Полный русский интерфейс
- 📊 Подробный лог всех операций (кто, что и когда удалилось)
- ⏸️ Прогресс-бар с процентами в реальном времени

**Как использовать:**
1. Нажми на иконку 🗑️ в тулбаре Discord
2. Заполни параметры: ID сервера, канала, автора (есть кнопка «получить» для автозаполнения)
3. Настрой фильтры по желанию
4. Нажми «Холостой запуск» для проверки, потом — «Запустить удаление»

### 🤖 **Social Copilot** — ИИ-анализ отношений в Discord (v2.0)

Твой личный психолог и социолог в Discord. Анализирует переписку с кем-то и рассказывает, как дела в этих отношениях.

**Что умеет:**
- 💫 **Vibe Check** — оценивает атмосферу переписки в процентах (0% — конфликт, 100% — идеал)
- 🧠 **Психологический портрет** — определяет тип привязанности человека, его черты характера
- 💬 **Живой чат с ИИ** — задавай вопросы про эту переписку, и Copilot будет отвечать в контексте
- 🔐 **Полная безопасность** — API ключи хранятся только у тебя локально в браузере
- 🎨 **Красивый UI** с фиолетовым свечением

**Как работает:**
1. Нажми иконку 🌟 в тулбаре Discord
2. Открой вкладку **«Настройки»** и вставь свой бесплатный API ключ от Google Gemini
3. Открой вкладку **«Вайб»** и нажми **«Запустить Vibe Check»**
4. Получи анализ переписки и перейди на **«Портрет»**, чтобы увидеть психопортрет
5. Чат в вкладке **«Чат с ИИ»** — задавай любые вопросы про эту переписку

**Где взять API ключ?**
- Заходишь на [Google AI Studio](https://aistudio.google.com/)
- Нажимаешь **«Create API Key»** → **«Create new free API key in new Google Cloud project»**
- Копируешь ключ и вставляешь в настройки скрипта
- Готово! Ключ хранится локально только в твоём браузере

## 🚀 Как установить оба скрипта?

### Шаг 1: Установи расширение для юзерскриптов
Выбери один из:
- **[Violentmonkey](https://violentmonkey.github.io/)** (работает везде)
- **[Tampermonkey](https://www.tampermonkey.net/)** (популярнее)

### Шаг 2: Установи скрипты
Для каждого скрипта (**Discord Mass Deleter.js** и **Social_Copilot_AI.js**):
1. Открой файл скрипта в репо
2. Нажми кнопку **Raw** (или **Скачать**)
3. Расширение автоматически предложит установить скрипт
4. Подтверди — и вот, скрипт работает!

### Шаг 3: Используй в Discord
Откройте Discord в браузере — в правом верхнем углу появятся две новые иконки:
- 🗑️ — Mass Deleter
- 🌟 — Social Copilot

## 💡 Советы

**Для Mass Deleter:**
- 📝 **Холостой запуск** — отличный способ проверить, что удалится, без риска
- 📊 Логи показывают каждое удалённое сообщение — если что, можно вручную восстановить
- ⚠️ **Архивные ветки** автоматически пропускаются (Discord их не даёт удалять)
- 🚫 **Системные сообщения** (приглашения, пины) тоже пропускаются

**Для Social Copilot:**
- 🔒 Твой API ключ 100% конфиденциален — хранится только в браузере, нигде не передаётся
- 📱 **Groupchat Mode** — если в чате больше 2 человек, ИИ анализирует групповую динамику, а не 1-на-1
- 💭 Чат запоминает историю разговора (в пределах одной сессии)
- 🔄 При свежем анализе (Vibe Check) каждый раз подгружаются последние 50 сообщений
